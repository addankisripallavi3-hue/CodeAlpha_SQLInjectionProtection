
# Secure Login System using Java

This project demonstrates a simple user registration and login system.

Features:
- User registration
- Password encryption
- Password decryption
- Login authentication
- Double-layer verification using username and password

Technologies Used:
- Java
- Eclipse IDE

Files:
1. AESUtil.java
2. RegisterUser.java
3. LoginUser.java
